How it works

  • GitHub and GitLab are set up separately on this page — each flow creates its own saved connection.
  • GitHub App: short-lived tokens and automatic repo webhooks.
  • GitLab OAuth: browser authorize; good for private clones — add webhooks from Deploy when needed.

1. Create GitLab Application

Setup steps

  1. Go to GitLab → Settings → Applications
  2. Name it anything, paste the callback URL above as Redirect URI
  3. Enable scopes: read_user api read_repository
  4. Save → copy Application ID & Secret → click Connect with OAuth
