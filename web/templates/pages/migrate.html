Restore on new VPS

      
Install NextDeploy on the target server first, then run migrate.sh with the download URL or a local bundle path.

    

    

      
curl -fsSL https://raw.githubusercontent.com/masudranaxpert/NextDeploy/main/migrate.sh | sudo bash -s -- \
  --url "https://your-panel.example.com/migrate/download/TOKEN"

      
Or with a local file: sudo bash migrate.sh --file /path/to/bundle.nd-migrate
