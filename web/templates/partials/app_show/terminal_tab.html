Terminal

Interactive TTY in the container over WebSocket (xterm.js + Docker exec). Pick a container — connection starts automatically.

Package manager depends on the image: Alpine → apk add …; Debian/Ubuntu → apt; Fedora/RHEL → dnf/yum. Alpine has no apt — "apt: not found" there is normal. If the status shows "Connection closed" immediately, check panel logs for terminal exec / terminal read docker.
